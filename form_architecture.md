# Формируем архитектуру
1. Схемы создания объектов - объекты будут создаваться в начале игры без дополнительной инфраструктуры.  
2. Схемы обарботки событий - код будет крутиться в цикле ожидая ввод игрока и проводя действия через паттерн State. Вызов отрисовки и начала запроса ввода будет в State.
3. Использование технологий - для отображения консоли будет использовать библиотека Textual.
3. Тесты https://github.com/mileoa/third_oop/tree/main/src/tests
