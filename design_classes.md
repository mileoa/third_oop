# Классы проектирования
Strategy - для разных типов специальных бонусов, бонусов и просто комбинаций  
State - для управления ходом игры  
NullObject - в комбинации со Strategy для обработки случаев когда нет специальных бонусов  
MVC - для обработки ввода, вывода, хранения информации о игре, обработки процесса игры
