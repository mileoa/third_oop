# Классы анализа
PlayingField -сетка 8x8, которая хранит элементы игрового поля и на которой происходят основыне действия игры.  
Statistic - статистика игры: очки, количество ходов.  
History - история ходов игрока.  
Console - принимает ввод и осуществляет вывод игровой информации.  
Gameplay - игровой процесс, отвечает за применение нужной последовательности действий в зависимости от текщуего состояния игры.  
Combination - комбинация, удаление которой дает определенное количество очков и может привести к использованию специального бонуса.  
SpecialBonus - специальнй бонус, применение которого меняет поведение игры.  
SpecialBonusActivation - применение специального бонуса.  
Error - ошибка во время игрового процесса.  
Player - игрок, взаимодействует с игрой.  
PatternMatching - поиск комбинаций.  
FieldElement - элемент игрового поля. Может быть разных типов  
GameOver - окончания игры.  
PlayerInput - ввод игрока.  
ElementsSwapping - перестановка элементов.  
ElementsGeneration - генерация элементов.  
ElementsDeletion - удаление элементов/комбинаций.
