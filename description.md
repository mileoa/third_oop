# Описание системы
Система реализует игру 3 в ряд. В системе есть  
1. Игровое поле - матрица 8x8 или NxM, которая хранит в себе элементы игрового поля. Игровое поле может инициализироваться, удалять элементы, менять элементы местами, сдвигать элементы сверху в пустые места снизу, создавать новые элементы в пустых местах, искать комбинпции.  
2. Игрок. Игрок может менять элементы местами, использовать специальные бонусы и хранить их, перезапукать игру, завершать игру. При перемене элементов местами ход игрока сохраняется в истории. Цель игрока собирать комбинации на игровом поле и зарабатывать очки.  
3. Специальный бонус - специальный эффект, который изменят поведение игры. Случайный специальный бонус выдается игроку при достижении каждых n очков. Специальный бонус может быть использован при достижении некоторых комбинаций. Игрок может использовать специальный бонус через ввод команды если есть доуступные специальные бонусы. Специальным бонусом можем быть удаление всех элементов определенного типа, удаление всего ряда и/или всей строки.  
4. Комбинация - расположение 3 игровых элементов одинакового типа по вертикали или горизонтали на игровом поле. При удалении дает игроку очки в зависимости от сложности комбинации. Также есть подтип комбинаций - бонусы, это сложные комбинации, например, расположение элементов в виде плюса. Бонусы дают больше очков чем комбинации.  
5. Игровой ход -  процесс в котором после изменения игроком местами элементов проверятся существование комбинаций, удаление комбинаций, применение специальных бонусов, заполнение поля пустыми элементами. Процесс заканчивается когда на поле не осталось комбинаций и поле полностью заполнено.  
6. Интерфейс - отображает игроку статистику игры в конце хода, ошибки ввода, игровое поле, специальные бонусы, принимать от игрока ввод для совершения доступных дейсвтий.
